### 安装linux

推荐安装ubuntu server 

### 配置Linux

1. 配置ssh

   ```shell
   sudo apt-get install ssh
   sudo service ssh start
   ```

2. 修改默认时区

3. 安装mysql

   ```shell
   sudo apt-get install mysql-server mysql-client
   ```

4. 安装必要工具（nettool等）

### SQL

设置mysql密码

导入 skydrive.sql

```shell
source  ./skydrive.sql
```

### 配置skydrive_debug

在可执行文件同级目录下复制config_ini.json，并配置

```json
{
  "mysqlDbUsername":"root",
  "mysqlDbPassword":"*******",
  "mysqlDbHost":"localhost",
  "mysqlDbPort":"3306",
  "mysqlDbName":"skydrive",
  "mysqlDbcharset":"utf8",
  "mysqlDbconSprintf":"%s:%s@tcp(%s:%s)/%s?timeout=%dms&readTimeout=%dms&writeTimeout=%dms&charset=%s",
  "HttpPORT":9996,
  "UDP_SERVER_ListenPORT":8997,
  "UDP_GroupSERVER_ListenPORT":8998,
  "UDP_GroupSERVER_SendPORT":8999,
  "UDP_SERVER_SendPORT":8996,
  "diskFileRoot":"updisk",
  "albumFileRoot":"upalbum",
  "thumbnailRoot":"thumbnail",
  "adminManagerDir":"html",
  "LogDir":"output/log"
}
```

### 安装ffmpeg

用于图片压缩和视频缩略图获取

```shell
sudo apt-get update

sudo apt-get install ffmpeg
```

### 设置开机自启动

编写启动脚本 start.sh 

```shell
#!/bin/bash
cd /home/wisn/
/home/wisn/skydrive_release &
exit 0 
```

添加到开机自启动 修改 /etc/rc.local

```shell
#!/bin/bash
#执行的命令
(
sleep 10
执行脚本  &
) &
exit 0
```
